Copy the contents of this zip directly into the existing web folder.

Included files:
- js/config.js: adds the Google OAuth client ID.
- js/screens/mainScreen.js: main desktop layout becomes 50:50 split, custom month modal, day click updates preview.
- js/screens/tripScreen.js: desktop trip screen gets a trip list/sidebar on the left.
- css/responsive.css: desktop/tablet split layout plus mobile fallback.

After copying:
1. Commit/push.
2. Wait for GitHub Pages.
